D3.js

D3.js介绍(翻译)

D3.js 是基于数据驱动文档工作方式的一款js函数库。你可以基于现有的HTML、SVG和CSS技术对数据进行可视化。D3.js注重对web标准的兼容，使你不用采用任何额外的框架，就可以在主流浏览器上使用D3.js。同时，D3.js 拥有强大的可视化组件和基于数据驱动文档的强大特性。

兼容性

除了IE8及以下，D3.js几乎支持所有的主流浏览器。D3.js 已在Firefox、Chrome、Safari、Opera、IE9+、Android和iOS上测试通过。只要有JavaScript 和 W3C DOM API，D3.js 的核心组件就可以运行，因此部分D3.js组件可以运行在老旧的浏览器上面。D3.js 使用Selectors API Level 1，但是你可以通过在插件Sizzle提供兼容性解决。你如果要使用SVG和CSS3 Transitions技术，你就需要一个“现代化”的浏览器了。
D3.js 并没有提供一个兼容性的解决方案，如果你的浏览器不是遵循web标准开发，那非常抱歉，你不能使用D3.js。

D3.js 同样可以运行在Node.js上面。通过npm install d3来进行安装，同时调用require("d3")来启用。在Node.js上面，你需要使用JSDOM提供一些有限的DOM支持。D3同样也可以配合WebWorker进行使用，但是你需要编译一个自定义仅包含你需要的功能（non-DOM）特性版本。
